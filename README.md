# Gestión de horas extras

## Información del proyecto

Proyecto frontend construido con Vite, React y TypeScript para la gestión de horas extras.

## Cómo ejecutar localmente

Requisitos: Node.js y npm/yarn.

```sh
# Clona el repositorio
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>

# Instala dependencias
npm install

# Inicia en modo desarrollo
npm run dev
```

## Tecnologías

- Vite
- TypeScript
- React
- Tailwind CSS

## Despliegue

Construye la app y despliega los archivos de `dist` en el hosting de tu preferencia:

```sh
npm run build
```
